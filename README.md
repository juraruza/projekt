## 00-Návrch a schválení vlastního tématu

## Název projektu
Simpsonovi hlavní postavy

## Žák
Jiří Růžička

## Stručný popis tématu
Webová stránka která bude zaměřena na hlavní postavy ze seriálu The Simpsons. Stránka bude mít podobu one-page webu s několika sekcemi které budou ukazovat záliby jednotlivých členu rodiny. Cílem projektu je abych vytvořil responzivní, přehledný web který se bude každému líbit.

## Cílová skupina 
Fanoušci seriálu The Simpsons který by se chtěli ponořit hlouběji do seriálu. 

## Návrch obsahových sekcí
1. O seriálu - kdy vznikl kdo je jejim autorem.
2. O autorovi - odkud pochází, jeho historie.
3. O hlavních postavách - věk, povolání, koníčky, atd.
4. Kontakt na autora webové stránky.

## Použité technologie
Frontend: HTML, CSS nebo Tailwind CSS, JavaScript
Data: JSON soubory pro čtyři textové sekce + JSON.
AJAX: načítání obsahu sekcí.
Php: pouze pokus neni jistý zda ho použiji.
Verzování: Git + GitHub (repozitář dostupný vyučujícímu)

## Očekávaný přínos / co se naučím
práce v javascriptu
práce s JSON a strukturovanými daty
načítání dat pomocí AJAX a dynamické vkládání obsahu
tvorba responzivního designu
základní formulářová komunikace se serverem
organizace projektu ve složkách, verzování přes GitHub
